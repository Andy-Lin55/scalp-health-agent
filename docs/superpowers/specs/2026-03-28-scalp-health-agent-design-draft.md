# 头皮健康 Agent — 设计草稿备份
> 保存时间：2026-03-28（设计进行中，非最终版）

---

## 基本定位

面向普通用户的头皮健康管理 Agent，帮助用户解决生发和头皮健康问题，并提供个性化方案推荐（包含产品及其他生活建议）。

---

## 平台与渠道

- 移动端 App（React Native / uni-app）
- Web 端（Next.js / Vue）
- 微信小程序（uni-app / 原生）
- 三端共用统一 REST / WebSocket API 网关

---

## 架构方案：方案 B — 带工具调用的模块化 Agent

一个主对话 Orchestrator Agent，按意图调用专属工具模块，用户感知为自然对话，后台结构化调用。

### 工具模块（5个）
1. 头皮诊断工具
2. 图片分析工具
3. 方案推荐工具（产品 + 饮食 + 习惯 + 成分补充 + 就医建议）
4. 健康档案工具
5. 跟进提醒工具

### AI 模型路由
- 对话推理：Claude API（主）/ 国内大模型（备，合规）
- 图片分析：专业视觉模型（GPT-4V / 自建）
- 产品匹配：Embedding 向量相似度

---

## 用户体验

### 交互模式
- 对话为主，问卷和照片上传自然融入对话过程
- 鼓励但不强制照片上传和问卷完成
- 问卷字段拆解为自然对话问句，用户无感建档
- 关键问题附快捷选项气泡（点选或文字均可）

### 首次用户流程
1. Agent 主动问候，询问主要困扰
2. 引导式对话建档（问卷问题 = 对话问题，照片邀请嵌入对话）
3. 信息逐步完善（每轮对话后档案自动更新）
4. 个性化方案输出（随时可给，信息越多越精准）

### 回访用户
- 自动加载历史档案，续接上次状态
- 追踪改善情况，对比历史记录
- 动态调整推荐方案

---

## 产品推荐逻辑

- 开放平台接入：京东 / 淘宝 / 小红书等电商 API
- 自有产品优先展示，提供更便捷的购买体验
- 基于 Embedding 向量相似度做产品匹配

---

## 数据与隐私

- 本地加密存储优先（用户设备）
- 可选云端账号同步（加密）
- 用户可授权贡献匿名加密数据用于模型迭代
- 数据同步由用户完全可控
